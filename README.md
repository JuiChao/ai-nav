# AI 导航 - 发现最好用的 AI 工具 (MiMo 美学复刻版)

精选收录各领域优质 AI 工具，涵盖对话、图像生成、视频创作、音频处理、编程开发、写作助手等，帮你快速找到提升效率的最佳方案。

本项目视觉与交互已完美复刻 `https://mimo.xiaomi.com/coder` 的极简纸面东方美学与标志性的鼠标水墨擦除背景图隐藏特效。

## ✨ 功能特性

- 🔍 **智能搜索** — 支持按名称、描述、标签搜索 AI 工具
- 📂 **分类导航** — 多大分类快速筛选（AI 对话、图像生成、视频创作等）

- 🎯 **SEO 优化** — 规范的单 `<h1>` 结构与 Open Graph 语义化标签
- 🎨 **MiMo 宣纸美学** — 极简东方留白、宋体书法字形与温润暖灰纸张感
- ✦ **鼠标水墨擦除** — **核心隐藏交互！** 鼠标移动时在遮罩层上擦出水墨边缘，显现底层宽幅飞天伎乐画卷，移开后平滑自愈

## 🎨 隐藏交互技术内幕

### 1. 32 段正弦波叠加水墨擦除算法
在 `CursorReveal` 组件中，鼠标移动时会将粒子（Stamp）存入活跃队列。在每帧渲染时，采用 `destination-out` 混合模式将 Stamp 擦出 Canvas。笔触由 32 分段圆周构成，每个角度的外径半径根据正弦方程抖动：
$$Radius_{wob} = R \times [0.78 + 0.14\sin(3\theta + \text{seed}) + 0.08\sin(7\theta + 2.1\text{seed}) + 0.05\sin(13\theta + 0.7\text{seed})]$$
配合径向渐变进行边缘羽化，完美模拟出宣纸上毛笔点染化开的随机毛刺墨迹。当队列空闲时，`requestAnimationFrame` 自动暂停，性能极其优异。

### 2. 卡片打字机与光标呼吸
在 `ToolCard` 中利用 `IntersectionObserver` 监听卡片进入视口的交叉比（超过 15% 时触发）。结合 React state 驱动字符逐个显现，打字期间光标保持常亮，播放完后转换为 1 秒周期的步进闪烁。

## 🎨 设计系统

- **配色方案**：宣纸暖灰 (`#FCFAF8`) 背景，墨黑 (`#26251E`) 文字，卡其细线 (`#E6E2DE`) 边框，温润淡雅。
- **字体规范**：中文字体选用宋体/明朝体（`Noto Serif SC`、`Songti SC`），英文字体为极简无衬线 `Questrial`。
- **Header 导航**：超薄极简顶栏，导航项悬停时底部黑线从左至右延伸展开，带有二级“产品”和“语言选择”悬停浮窗。
- **卡片底色**：卡片背景随机/循环由五种柔和暖调色卡（`#EFEBE3`、`#F6F1EA`、`#EFE8E3` 等）组成，Hover 时卡片浮起，反转为纯白 `#FFFFFF`，外链图标滑入，动感十足。
- **页脚 Footer**：极简居中多链接排版，带有细分割线 `|` 装饰。
